SEMANA 01

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar contenido en la web. Es un lenguaje
de marcado que utiliza etiquetas para definir la estructura y presentación de una página web.En HTML, he aprendido los siguientes conceptos y habilidades: Estructura básica: HTML se compone de una estructura básica que incluye etiquetas como html, head, y body. Estas etiquetas
definen la estructura general de una página web.Etiquetas y elementos: HTML utiliza etiquetas para marcar diferentes elementos en una página web, como encabezados , párrafos , t enlaces , imágenes, lista, entre otros.Atributos: Las etiquetas pueden tener atributos que proporcionan información adicional. Por ejemplo, el atributo href se utiliza en
la etiqueta para especificar la URL de destino de un enlace.Anidamiento: Los elementos HTML se pueden anidar dentro de otros elementos, lo que permite crear una estructura jerárquica y organizar
el contenido de manera adecuada. Formularios: HTML proporciona elementos de formulario form que permiten a los usuarios ingresar y enviar datos, como campos de texto input
type="text", botones button, casillas de verificación input type="checkbox", entre otros.Imágenes y multimedia: HTML permite la inclusión de imágenes img y contenido multimedia, como videos y audio, utilizando etiquetas específicas. Tablas: HTML ofrece etiquetas para crear tablas table y definir filas tr, columnas td, encabezados th, etc. Etiquetas semánticas: HTML5 introduce etiquetas semánticas como header, nav, section, article, entre otras, que ayudan a estructurar el contenido
de manera más significativa para los motores de búsqueda y la accesibilidad. Enlaces: HTML permite crear enlaces a que direccionan a otras páginas web o a secciones específicas dentro de la misma página mediante el uso del atributo href. Estos son solo algunos de los conceptos y habilidades que he aprendido en HTML. Con esta base, puedo crear y estructurar contenido en la web,
aprovechando todas las etiquetas y elementos disponibles para diseñar páginas web interactivas y atractivas.

PRIMERA TAREA

Una de las principales características de HTML es su estructura basada en etiquetas, que se utilizan para marcar diferentes elementos en una página web. Estas etiquetas proporcionan instrucciones al navegador sobre cómo interpretar y mostrar el contenido.
