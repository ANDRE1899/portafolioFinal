SEMANA 04

MANEJO DEL DOM CON JAVASCRIPT

El manejo del DOM con JavaScript ha sido una parte fascinante de mi trayectoria en el desarrollo web.
A través de este lenguaje, he aprendido a utilizar métodos y propiedades para interactuar con los elementos del DOM y modificar
su contenido, estilos y atributos de manera dinámica. Además, he explorado el uso de animaciones para crear efectos visuales
atractivos y mejorar la experiencia del usuario. Uno de los proyectos destacados en este campo ha sido el desarrollo de un juego
utilizando JavaScript, donde he aplicado mis conocimientos para crear una experiencia interactiva y entretenida. Estoy emocionado
por seguir ampliando mis habilidades en el manejo del DOM y explorar nuevas formas de utilizar JavaScript para desarrollar
proyectos web innovadores. Se desarrollo al final de esta sesión el juego de una pelota que puede pausarse , cambiar el sentido de movimiento y reiniciar
este movimiento
