SEMANA 14

API REST

A lo largo de mi aprendizaje en el desarrollo web, he tenido la oportunidad de sumergirme
en el apasionante mundo de las API REST utilizando Java y Tomcat. Al principio, el proceso de configurar el entorno
y entender cómo funcionaba cada componente me pareció un desafío, pero con dedicación y perseverancia, he logrado
adquirir conocimientos valiosos en esta área. Gracias a mi dedicación, pude configurar correctamente mi entorno de desarrollo, asegurándome de tener instalado Java
JDK y descargando Apache Tomcat para desplegar mis aplicaciones web. A medida que avanzaba en mi aprendizaje, fui
desarrollando mi aplicación Java y definiendo las entidades y modelos que representarían los datos manejados por mi
API REST. La parte más emocionante fue la implementación de los controladores, ya que a través de ellos pude manejar las solicitudes
HTTP y diseñar la lógica de negocio para procesar y responder a cada petición. Utilizando las anotaciones adecuadas, definí
las rutas de mi API REST y logré que mis controladores interactuaran con las entidades y modelos para satisfacer las
necesidades de los clientes.
