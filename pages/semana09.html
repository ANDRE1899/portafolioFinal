ENRUTAMIENTO Y CONSUMOP API

A lo largo de mi aprendizaje en el desarrollo web, he descubierto la importancia del routing o enrutamiento
en la creación de aplicaciones web. Este concepto se refiere a la forma en que las aplicaciones web gestionan las solicitudes de los
usuarios y toman decisiones sobre qué acción ejecutar en función de la URL que se está solicitando. Para que mis aplicaciones sean interactivas y eficientes, es fundamental entender cómo el enrutamiento juega un papel clave en este
proceso. Cuando un usuario interactúa con mi sitio web, cada clic o solicitud que realiza llega a través de una URL específica. Aquí
es donde el enrutamiento entra en acción, ya que es responsable de interpretar esas URLs y dirigirlas al componente o controlador
adecuado para su procesamiento y respuesta. En otras palabras, el enrutamiento actúa como un mecanismo de asignación entre las URLs y los diferentes componentes o controladores
que he creado previamente. Cada componente o controlador está diseñado para manejar una determinada solicitud, lo que me permite
organizar y gestionar eficientemente las interacciones de los usuarios con mi aplicación web
